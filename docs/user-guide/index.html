Here are a few key things to know before you get started:

  • Tiers: Every user is assigned the free tier by default which has certain restrictions unlike the Pro tier. You can view Pro tier benefits and upgrade your tier at any time from the main menu.
  • Fees: This bot applies a 1% fee on all trades for free tier users. Fees are collected automatically from your wallet when the balance owed exceeds a certain threshold. Pro tier users have no fees.

  • Help: If you need assistance, access the help menu from the main menu or use the /help command at any time.

2. Getting Started

2.1. First-Time Setup

When you first /start the bot, it will guide you through a setup process:

  1. Language Selection: Choose your preferred language.
  2. Wallet Setup: The bot will help you set up both an EVM (Ethereum, BSC, etc.) and a Solana wallet. For each, you can:
    • Import Wallet from Private Key: Use an existing wallet.
    • Import Wallet from Seed Phrase: Import from a 12 or 24-word seed phrase.
    • Generate New Wallet: Create a fresh, empty wallet.
  3. Telegram API Setup: To use the automated snipers (MultiSniper and SolanaSniper), you must add your Telegram API credentials via the settings menu. This allows the bot to read messages from channels you specify. Go to Settings -> MultiSniper Settings (or SolanaSniper Settings) -> Manage Telegram Settings.

2.2. How to Start Sniping

EVM (Ethereum, BSC, Base etc.): There are two bots available:

  • LaunchSniper: Use this bot to snipe a specific token as soon as it launches or gains liquidity.
  • MultiSniper: Automatically buy and sell tokens from configured Telegram channels.
To start sniping on any EVM chain, follow these 5 simple steps:
  1. Use /start to go to the main menu.
  2. Click on 'Change Chain / DEX', and switch to your desired EVM chain.
  3. Fund your wallet with the native currency (e.g. ETH, BNB). You can view your wallet address via 'Wallet Info' in the main menu.
  4. Go into the settings menu (/settings) to configure the LaunchSniper or MultiSniper.
  5. Start the bot from the main menu or using the /launch or /multi commands.

Solana: You can use the SolanaSniper bot to automatically find and trade tokens from Telegram channels.
To start sniping on Solana, follow these 5 simple steps:

  1. Use /start to go to the main menu.
  2. Click on 'Change Chain / DEX', and switch to Solana.
  3. Fund your wallet with SOL. You can view your wallet address via 'Wallet Info' in the main menu.
  4. Go into the settings menu (/settings) to configure the SolanaSniper.
  5. Start the bot from the main menu or using the /solana command.

3. Main Menu Explained

The main menu is your central hub for controlling the bot.

  • Snipe token (LaunchSniper) (EVM): Starts the LaunchSniper bot to buy a specific token on an EVM chain. You will be prompted to enter the token's contract address.
  • Auto buy / sell (MultiSniper) (EVM): Starts the MultiSniper bot for EVM chains, which automatically finds and trades tokens from your configured Telegram channels.
  • Auto buy / sell (MultiSniper) (Solana): Starts the SolanaSniper bot, which automatically finds and trades tokens from your configured Telegram channels on the Solana network.
  • SOL Reclaimer: A utility to recover SOL from empty token accounts in your Solana wallet, helping you clean up dust and reclaim locked SOL.
  • Snipe List: Shows a history of your past snipes and trades.
  • Wallet Info: Displays your currently active wallet address (both EVM and Solana) as a QR code and text, along with your native token balance (e.g., ETH, BNB, SOL). You can also reveal your private key from this menu.
  • Change Chain / DEX: Allows you to switch between different blockchains (e.g., Ethereum, BSC, Solana) and the decentralized exchanges (DEXs) on them.
  • Settings: Opens the detailed configuration menu, covered in section 5.
  • Help: Displays the help message.
  • Referral Scheme: Opens the menu to view your referral statistics and get your unique referral link.
  • Pro Tier: Shows the benefits of the Pro Tier and allows you to upgrade your subscription.

4. Commands

  • /start: Initializes the bot and displays the main menu.
  • /settings: Opens the main settings menu.
  • /help: Displays the help message.
  • /commands: Shows a list of available commands.
  • /launch: Starts the LaunchSniper for EVM chains.
  • /multi [y]: Starts the MultiSniper for EVM chains. Add y to auto-start.
  • /solana [y]: Starts the SolanaSniper. Add y to auto-start.
  • /reclaim: Starts the SOL token reclaimer utility.
  • /sell: Lists currently held tokens to sell them manually.
  • /sell [percentage]: Sells a specific token from the list by its ID.
  • /sell all [percentage]: Sells all tokens currently held.
  • /sell , [percentage]: Sells multiple specific tokens.
  • /sell [percentage]: (Solana Only) Sells a token by its contract address.
  • /list: Shows your snipe history.
  • /feesummary: Displays your trading fee summary.
  • /profits: Shows your profit/loss summary.
  • /refer: Opens the referral menu.
  • /stats: Shows public statistics for the bot.
  • /clearcache: Deletes the token cache file for your active sniper.
  • /clear: Clears the message history.

5. Configuration Settings

This section details all settings available in the /settings menu.

5.1. EVM Settings

Wallet Settings (EVM)

  • Show Private Key: Displays your current EVM wallet's private key.
  • Generate New Wallet: Creates a new EVM wallet, overwriting the old one.
  • Import Existing Wallet: Overwrites your current wallet by importing a new one via private key.

Gas Settings (EVM)

  • Set Transaction Type: Choose between Legacy (for BSC, etc.) and EIP-1559 (for Ethereum).
  • Legacy TX Settings:
    • Set Buy/Approve/Sell Gas Price: The gas price in Gwei.
    • Set Buy/Approve/Sell Gas Limit: The maximum gas the transaction can use.
    • Toggle Auto Gas Price Calculation (Pro): Automatically sets the gas price based on network conditions.
  • EIP-1559 TX Settings:
    • Set ... Max Priority Fee: The "tip" to the validator in Gwei.
    • Set ... Max Base Fee: The maximum base fee you're willing to pay.
    • Set ... Gas Limit: The gas limit for the transaction.
    • Toggle Auto Gas Price Calculation (Pro): Automatically sets fees.
    • Set Auto Gas Price Priority: Sets the speed for auto-calculated fees (Slow, Medium, Fast).

LaunchSniper Settings (EVM)

  • Transaction Revert Time: How long to wait (in seconds) before considering a transaction failed.
  • Slippage Percentage: The maximum percentage the price can change before your transaction reverts.
  • Basic Autosell Multipliers: Set the take-profit (e.g., 2 for 2x) and stop-loss (e.g., 0.5 for -50%) multipliers.
  • Sell After Time Delay (Blocks): The number of blocks to wait after a buy before automatically selling.
  • Trailing Stop Loss (TSL) Settings (Pro): Configure the TSL (see section 6.4).
  • Max Buy/Sell Fee (Trading Checker): The maximum tax percentage the bot will tolerate when buying or selling a token.

MultiSniper Settings (EVM)

This opens a sub-menu:

  • Manage Trading Settings: Configure the core trading parameters (see below).
  • Manage Telegram Settings: Set up your Telegram API ID and Hash to allow the bot to read channel messages.
  • Manage Monitored Channels: Add, edit, or delete the Telegram channels you want the bot to monitor. You can set whitelist/blacklist keywords for each channel (Pro feature).
  • Manage Liquidity Pairs: Configure which tokens the bot should use for liquidity (e.g., WBNB, USDC) and set buy amounts and liquidity filters for each.

MultiSniper Trading Settings:

  • Toggle Buy/Sell Tokens: Master switches to enable or disable trading.
  • Set Trading Mode: Choose your automatic selling strategy (MANUAL_SELLING, BASIC_AUTOSELL, SELL_AFTER_TIME_DELAY, TRAILING_STOP_LOSS (Pro)).
  • Set Total Allowed Snipes: The max number of buys before the bot stops (-1 for unlimited).
  • Toggle Allow Buy Previously Owned Tokens: If enabled, the bot can re-buy tokens it has sold before.
  • Toggle Use Single Liquidity Pair: If enabled, the bot only buys tokens paired against the chain's native wrapped asset (e.g., WBNB, WETH).
  • Toggle Enable Trading Checker: Enables pre-buy safety checks for taxes and honeypots.
  • Set Auto Sell Percentage: The percentage of a position to sell when an auto-sell condition is met (e.g., 100 for all).

5.2. Solana Settings

Trading Settings (Solana)

This menu contains all the core trading parameters for the SolanaSniper, mirroring the EVM MultiSniper settings but with some Solana-specific adjustments.

  • Toggle Buy/Sell Tokens
  • Set Trading Mode: (MANUAL_SELLING, BASIC_AUTOSELL, SELL_AFTER_TIME_DELAY, TRAILING_STOP_LOSS (Pro)).
  • Set Total Allowed Snipes
  • Toggle Allow Buy Previously Owned Tokens
  • Simulation Mode:
    • Toggle Simulation Mode: If enabled, the bot will simulate buys and sells without using real funds.
    • Set Simulation Starting Balance: Sets the virtual SOL balance for simulation mode.
  • Buy Settings:
    • Set Buy Amount (SOL): The amount of SOL to use for each purchase.
    • Set Slippage Percentage
  • Sell Settings:
    • Set Auto Sell Percentage
    • Set ... (Basic Autosell)
    • Set Seconds Delay (Sell After Time Delay): Note this is in seconds, not blocks.
    • Set ... (TSL) (Pro)

Wallet, Gas, Node, and Telegram Settings (Solana)

  • Manage Wallet Settings: Import or generate your Solana wallet.
  • Manage Gas Settings:
    • Set Priority Fee (SOL): The extra fee to add to transactions for faster processing.
    • Set Compute Unit Limit: The maximum compute units a transaction can use.
  • Manage Node Settings (Pro): Set a custom Solana RPC node URL.
  • Manage Telegram Settings: Same as the EVM MultiSniper.
  • Manage Monitored Channels: Same as the EVM MultiSniper.

5.3. Other Settings (General)

  • Set Price Difference Alert Threshold: The percentage price change required to trigger a new price update message in your chat (e.g., 0.1 for 10%).
  • Keep Price Message History: If disabled, new price updates will edit the previous message instead of sending a new one.
  • Change Language
  • Reset All Settings: Wipes your user data file and resets the bot to its default state. Use with caution!

6. Trading Modes Explained

6.1. Manual Selling

The bot buys a token and holds it until you manually sell it using the /sell command.

6.2. Basic Autosell

The bot sells automatically when the token's price hits either a take-profit or a stop-loss target.

Example: You set Auto Sell Multiplier to 3 (for 3x) and Stop Loss Multiplier to 0.5 (for -50%). If the token's value triples, the bot sells. If it drops to half its initial value, the bot sells.

6.3. Sell After Time Delay

The bot sells automatically after a fixed amount of time has passed.

  • EVM: Set in blocks.
  • Solana: Set in seconds.

6.4. Trailing Stop Loss (TSL) (Pro Feature)

TSL is a powerful strategy that lets your profits run while protecting you from sharp price drops.

How It Works

  1. Activation: The TSL does nothing until the token's price reaches your Min Activation multiplier.
  2. Tracking the Peak: Once activated, the bot remembers the highest price multiplier the token reaches (the "peak").
  3. Setting the "Trap": The bot calculates a "trail price" that is a certain percentage below the peak. This percentage is your Trail Amount.
  4. Triggering the Sell: If the price drops from its peak and hits the trail price, the bot sells immediately. If the price keeps rising, the trail price moves up with it, always staying the same percentage below the new peak.

TSL Example

  • Your Settings:
    • Min Activation: 1.5x
    • Trail Amount: 0.2 (which is 20%)
    • Hardcap: 10x (An optional safety net to sell at a fixed high point)

The Trade:

  1. You buy a token for $100.
  2. The price goes to $140 (1.4x). TSL is not active yet.
  3. The price hits $150 (1.5x). TSL is now active. The peak is 1.5x.
  4. The trail price is set at 1.5x * (1 - 0.2) = 1.2x. If the price drops to $120, the bot will sell.
  5. The price continues to rise to $300 (3.0x). The peak is now 3.0x.
  6. The trail price is recalculated and moves up to 3.0x * (1 - 0.2) = 2.4x.
  7. The price then drops to $240 (2.4x). The bot sells immediately, locking in a ~2.4x profit.

This strategy allows you to capture the majority of a pump while automatically protecting your gains when the trend reverses.
